Mission goal
============

The Royal Spy has hidden crucial information in what appears to be a simple text file 
in the Safe. However, something seems off about this file.

Your mission is to:
1. Determine the real nature of this suspicious file
2. Extract the hidden information it contains

Useful commands
==============

file FILE
    Determines the type of FILE by examining its contents

exiftool FILE
    Reads and displays metadata from FILE (images, PDFs, etc.)
