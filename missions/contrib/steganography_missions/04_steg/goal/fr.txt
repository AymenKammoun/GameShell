Objectif de la mission
============

L'espion royal a caché des informations cruciales dans ce qui semble être un simple fichier texte 
dans le Safe du roi. Cependant, quelque chose ne tourne pas rond dans ce fichier.

Votre mission consiste à inspecter le fichier suspect pour retrouver le message caché de l'espion

Commandes utiles
==============

file FICHIER
    Détermine le type de FICHIER en examinant son contenu

exiftool FICHIER
    Lit et affiche les métadonnées de FICHIER (images, PDF, etc.)
