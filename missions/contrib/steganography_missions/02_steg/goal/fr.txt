Objectif de la mission
============

Après avoir découvert le premier traître, l'espion du roi a mis au jour une autre 
complot ! Une fois de plus, il a caché le nom d'un deuxième traître dans un nouveau tableau. 
nouveau tableau. Mais cette fois, il a été plus prudent...

Il a protégé le message caché par une phrase d'authentification et a stocké cette phrase d'authentification 
dans une note à l'intérieur du coffre-fort du roi. Les permissions de la note ont été définies pour 
pour empêcher toute lecture occasionnelle.

Votre mission consiste à :
1. Trouver et lire la note protégée dans le coffre du roi.
2. Utiliser son contenu pour extraire le message caché du nouveau tableau.
3. Révéler le nom du second traître

Commandes utiles
==============
chmod [OPTIONS] FICHIER
     Modifie les permissions du fichier (ou du répertoire).

eog IMAGE
    Ouvre un aperçu de l'IMAGE sous Linux

COMMANDE &
  Exécute la commande en arrière-plan.

steghide extract -sf FICHIER -p PASSPHRASE
    Extrait les données cachées du FICHIER en utilisant la PASSPHRASE utilisée pour intégrer les données dans le FICHIER. 
    dans le FICHIER.
